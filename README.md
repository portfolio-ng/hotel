# Hotel business contract

## commcomp/
##### Commissioned to design custom software to determine if booking companies were over charging for commissions based on the property management system data
*  Read in data from the three unique online sources
  * Normalise Data
  * Cleanse Data
  * Identify Outliers in Data
* Compare Values
  * Ensure the Dates, Booking Identification Number, Cost, And Guest Names matched the PMS where commissions were charged
  * Report any discrepancies
